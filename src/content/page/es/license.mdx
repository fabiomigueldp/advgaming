## Atribución Legal para Plantillas Gratuitas

[Unfolding.io](Unfolding.io) proporciona plantillas de sitios web profesionalmente diseñadas y optimizadas para dispositivos móviles. Cuando descargas o usas nuestras plantillas sin pagar, se aplican las siguientes condiciones. Lo que significa:

### Está permitido:

- Puedes usar las plantillas para uso personal y comercial.
- Puedes personalizar las plantillas como desees.
- Puedes convertir/adaptar las plantillas HTML para usar con cualquier CMS.

### Limitaciones para versiones GRATUITAS:

- No está permitido eliminar el crédito de copyright del pie de página.
- No está permitido vender, revender, alquilar o arrendar nuestras plantillas.
- No proporcionamos soporte técnico para plantillas HTML gratuitas.

**Si deseas eliminar la atribución, por favor adquiere una [licencia](https://nebulix.unfolding.io/shop/starfunnel-license/)**
