      ## Guia Rapido de Instalacao

      ### OBS Studio

      1. Baixe e extraia o kit em uma pasta organizada
      2. Abra o OBS Studio
      3. Para overlays: Adicione uma fonte de "Imagem" e selecione o arquivo
      4. Para alertas: Adicione uma fonte de "Navegador" e configure a URL do widget
      5. Posicione e redimensione conforme necessario

      ### Streamlabs

      1. Abra o Streamlabs Desktop
      2. Va em "Editor" e selecione a cena desejada
      3. Clique em "+" para adicionar fontes
      4. Selecione "Imagem" para overlays ou "Fonte de Navegador" para widgets
      5. Importe os arquivos do kit

      ### Dicas importantes

      - Mantenha os arquivos do kit em uma pasta que nao sera movida
      - Faca backup dos arquivos originais antes de editar
      - Use as resolucoes recomendadas para cada elemento
